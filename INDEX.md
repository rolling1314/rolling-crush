## 📝 代码文件

### 后端核心文件
```
crush-main/internal/
├── auth/
│   ├── jwt.go          - JWT 生成和验证
│   ├── user.go         - 用户管理
│   └── middleware.go   - HTTP 认证中间件
├── httpserver/
│   └── server.go       - HTTP API 服务器
├── server/
│   └── server.go       - WebSocket 服务器
├── app/
│   └── app.go          - 应用主类
└── cmd/
    └── root.go         - 启动配置
```

### 前端核心文件
```
crush-fe/src/
├── components/
│   └── LoginPage.tsx   - 登录页面
├── App.tsx             - 主应用
├── types.ts            - 类型定义
└── main.tsx            - 入口文件
```
